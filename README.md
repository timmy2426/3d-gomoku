# 3d-gomoku
3D五子棋是一款將平面棋盤立體化的創新五子棋遊戲。不同於傳統的平面棋盤，立體棋盤不受邊界限制，創造出獨特的盤面特性，為玩家帶來全新的遊戲體驗。
